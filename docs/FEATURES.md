# 功能概览

## 核心功能

### 工作流引擎
- 工作流生命周期管理（创建→启动→执行→完成/失败）
- 消息驱动架构
- 中断与取消支持
- 异步执行（`std::future`）

### 工作流编排
- DAG（有向无环图）编排
- 并行执行与串行执行
- 循环依赖检测
- 节点级超时控制
- 执行状态追踪

### 持久化
- SQLite 数据库存储
- 工作流状态快照
- 执行历史记录
- 成功率统计

### 调度
- 一次性定时任务
- 周期性调度
- Cron 表达式支持
- 调度取消

### 版本控制
- 工作流版本创建
- 版本回滚
- 版本对比

### 模板
- 参数化工作流模板
- 模板实例化
- 参数类型校验

## 可靠性保障

### 熔断器（Circuit Breaker）
- 三态：关闭→打开→半开
- 可配置失败阈值
- 自动恢复探测
- 状态变更回调

### 重试策略
- 固定延迟重试
- 指数退避重试
- 最大重试次数
- 可重试条件判断

### 超时处理
- 全局/节点级超时
- 超时策略配置

### 死信队列
- 失败任务入队
- 重试机制
- 自动清理过期任务

## 监控与可观测

### 观察者模式
- 工作流启动/完成/中断/错误事件
- 自定义回调

### 指标收集
- 执行次数、成功率、耗时统计
- 自动统计报告

### 追踪
- 执行事件记录
- 时间线追踪

### 告警
- 规则引擎
- 级别分类（Info/Warning/Critical）
- 告警状态管理

### 日志
- 同步日志（单例）
- 异步日志（后台线程）
- 多级别（Debug/Info/Warning/Error）

## 性能

### 内存池
- 固定大小预分配
- O(1) 分配/释放
- 自动扩容
- 内存上限控制
- 线程安全

### 对象池
- RAII 智能指针自动归还
- 无需手动管理

## 配置
- JSON 配置文件
- 类型安全读写
- 嵌套配置支持

## 检查点
- 工作流状态快照
- 可恢复检查点
- 导入/导出

## 条件工作流
- 分支条件判断
- 多分支支持
- 默认分支

## 插件系统
- 动态加载/卸载
- 依赖解析
- 服务注册
- 消息总线
